"Псевдо ИИ" на Python для Telegram чата

Demetrius Bellioz 19 июля 2020 Python

У нас с друзьями есть групповой чат в Telegram. Дабы разнообразить наше общение, я решил написать бота который случайным образом пишет сообщения в чат. В основе своих сообщений бот использует цепь Маркова и историю сообщений чата.

Пример небольшого диалога с участием бота:

Краткое описание алгоритма

Находясь в групповом чате Telegram, бот записывает каждое сообщение в лог. Одновременно с этим берет случайное слово из сообщения и производит поиск строк по логу в которых есть это слово. Из найденных строк формирует сообщение, используя цепь Маркова, и отправляет его в чат.

Чтобы ограничить размер потребляемой оперативной памяти и места на диске, установлен лимит лога в 10000 последних сообщений. При превышении лимита будет происходить ротация лога.

Настройка

  1. Создать бота в @Botfather. Полученный токен прописать в переменную API_TOKEN.
  2. В @Botfather отключить режим конфиденциальности бота (Bot Settings -> Group Privacy -> Turn Off).
  3. Установить библиотеки из requirements.txt (pip install -r requirements.txt).
  4. Запустить бота и добавить в групповой чат.

Команды бота

Бот обрабатывает сообщения в зависимости от настроек /set_level. Например если /set_level равен 50, то бот будет обрабатывать каждое новое сообщение с вероятностью в 50% , если /set_level равен 100, то будет обрабатывать абсолютно каждое новое сообщение. По умолчанию /set_level равен 10.

При добавлений бота в чат, создаются файлы лога (logs/ID_чата.log) и настроек этого чата (configs/ID_чата.cfg). Если по какой-то причине этого не произошло, то можно воспользоваться командой /reset. Эта команда доступна только администраторам чата.

Версия Python и используемые библиотеки

  • Python 3.9.1
  • Aiogram
  • Markovify
  • Nltk
  • Configparser
