Автозапуск Python-скрипта в Tmux после перезагрузки Linux сервера

Demetrius Bellioz 22 сентября 2020 Linux

Задача: после перезагрузки сервера автоматически запустить терминал Tmux, и выполнить в нем Python-скрипт.

Нам понадобятся:

  1. Командный интерпретатор Bash
  2. Планировщик задач Cron

Действия:

  1. Создадим Bash-скрипт с именем Start.sh, в котором пропишем команды запуска терминала Tmux с заданными параметрами. 


    #!/bin/bash
    # Создаем новую сессию Tmux с именем MySession.
    tmux new-session -d -s MySession
    # Тайм-аут 10 секунд.
    sleep 10
    # Переходим в директорию где находится наш Python-скрипт.
    tmux send-keys -t MySession "cd /home/Python" C-m
    # Тайм-аут 10 секунд.
    sleep 10
    # Запускаем Python-скрипт с именем Script.py.
    tmux send-keys -t MySession "python3 Script.py" C-m

  2. Не забываем сделать наш Bash-скрипт исполняемым, в директории с файлом выполните команду:


    chmod +x Start.sh

  3. Добавляем запуск нашего скрипта в автозагрузку:


    Открываем файл crontab при помощи команды crontab -e, и в самом конце добавляем строчку @reboot /home/Python/Start.sh; exit

Готово! Теперь после каждой перезагрузки сервера будет запускаться наш скрипт Start.sh. Который в свою очередь будет запускать терминал Tmux, и передавать ему команды на запуск Python-скрипта.

P.S. Путь /home/Python, имя сессии MySession, и имя скрипта Script.py можете поменять на свои.
