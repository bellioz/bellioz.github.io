{"posts":[{"title":"Установка и настройка Fail2ban","content":" Fail2ban анализирует логи служб, работающих на сервере, и при обнаружении подозрительной активности блокирует IP-адреса возможных злоумышленников. Установка sudo apt update sudo apt install fail2ban -y Добавление в автозагрузку sudo systemctl enable fail2ban Настройка Настройки хранятся в конфигурационном файле: /etc/fail2ban/jail.conf Рекомендуется вносить свои настройки в файл jail.local, разместив его в той же директории. Создание файла jail.local: sudo touch /etc/fail2ban/jail.local Открываем файл jail.local и прописываем настройки: sudo nano /etc/fail2ban/jail.local Пример настроек: [DEFAULT] ignoreip = 127.0.0.1 [sshd] enabled = true findtime = 86400 maxretry = 3 bantime = 86400 ignoreip - Белый список IP-адресов, которые не будут блокироваться, указываются через пробел. [sshd] - Указание службы для которой применяются настройки. enabled - Включение (true) или отключение (false) работы Fail2ban с данной службой. findtime - Период времени (секунды), в течении которого регистрируются неудачные попытки авторизации. maxretry - Количество неудачных попыток авторизации. bantime - Продолжителность бана IP-адреса (секунды). После сохранения настроек перезапускаем Fail2ban: sudo systemctl restart fail2ban.service Просмотр логов Fail2ban: sudo tail /var/log/fail2ban.log ","link":"https://bellioz.github.io/post/ustanovka-i-nastroyka-fail2ban/"}]}
